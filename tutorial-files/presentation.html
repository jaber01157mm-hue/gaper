👤 الشخص الأول: فكرة المشروع وواجهة المستخدم (React)

الكلام اللي هيقوله:

"السلام عليكم، مشروعنا هو AutoPro، منصة ذكية لحجز وإدارة خدمات صيانة السيارات. هدفنا كان تحويل عملية الحجز اليدوية لنظام رقمي سهل وسريع."

"بالنسبة للواجهة الأمامية (Frontend)، استخدمنا مكتبة React.js مع بيئة عمل Vite عشان تكون سريعة جداً. قسمنا الموقع لمكونات (Components) زي Navbar و BookingForm عشان نقدر نعدل على أي جزء بسهولة من غير ما نأثر على باقي الموقع."

كود للشرح:

// مثال على Component في React
export default function Navbar() {
  return (
    <nav>
      <h1>AutoPro</h1>
      <button>تسجيل الدخول</button>
    </nav>
  );
}

👤 الشخص الثاني: التصميم والتجاوب (Tailwind CSS)

الكلام اللي هيقوله:

"التصميم كان جزء مهم جداً عشان راحة المستخدم. استخدمنا إطار عمل Tailwind CSS، وده بيخلينا نكتب التصميم جوه الـ HTML مباشرة بخواص جاهزة بدل ما نكتب ملفات CSS ضخمة."

"كمان الموقع متجاوب بالكامل (Responsive)، يعني بيشتغل بشكل مثالي على الموبايل والكمبيوتر. واعتمدنا على اللون الأسود والبرتقالي عشان يدي طابع احترافي خاص بالسيارات، وضفنا أنيميشن ناعم باستخدام مكتبة Motion."

كود للشرح:

<!-- مثال على استخدام Tailwind -->
<button class="bg-orange-500 hover:bg-orange-600 text-white font-bold py-2 px-4 rounded-xl transition-all">
  احجز الآن
</button>

👤 الشخص الثالث: قواعد البيانات والمصادقة (Supabase)

الكلام اللي هيقوله:

"عشان نحفظ بيانات المستخدمين والحجوزات، استخدمنا Supabase (وهو بديل قوي لـ Firebase بيعتمد على قواعد بيانات PostgreSQL)."

"عملنا جدولين أساسيين: جدول للمستخدمين (Users) وجدول للحجوزات (Bookings). وفعلنا نظام المصادقة (Auth) عشان المستخدم يقدر يسجل دخول بحساب جوجل بتاعه بأمان، وربطنا ده بنظام RLS عشان كل مستخدم يشوف حجوزاته هو بس."

كود للشرح:

// مثال على إضافة حجز جديد في قاعدة البيانات
const { error } = await supabase.from('bookings').insert({
  userId: user.id,
  vehicleMake: 'BMW',
  serviceType: 'oil_change',
  status: 'pending'
});

👤 الشخص الرابع: دمج الذكاء الاصطناعي (Gemini AI)

// دمج الذكاء الاصطناعي لتحليل المشكلة
